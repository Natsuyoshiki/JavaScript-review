Arrays is another type of values that represents a list of values
         [x, y, etc..] array can be turned into a variable
      array[0] selects the first value of the array, starts at 0
      array[1] = x;  changes value
    we can put any type of value inside an array, number, string, boolean, object, another array
    arrays have properties and methods : .length   .push()  .splice()

    loops (let us run some code over and over)

    while vs for loop
    standard loop => for  shorter and more organized
    non-standard loop => while

    Looping through an array
    1. go through each value of an array one by one
    2. Do something with each value

    accumulator pattern:
    calculate the total of all numbers in an array
    1.create a variable to store the result
    2.loop through the array and update the result

    arrays are references, same as object
    to make a copy of an array you can .slice(); (not splice)

    loops:
    1. break = exit a loop early
    2. continue
